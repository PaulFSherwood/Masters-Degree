# Week 1 — Algorithms and Big-O

WGU’s Applied Algorithms course emphasizes **Big O, benchmarking, profiling, data structures, algorithm design, implementation, and optimization**.

### Week 2 — Formal Languages and Programming Language Theory

WGU’s Formal Languages course covers **formal semantics, type systems, imperative/functional/parallel languages, compiled/interpreted/query/assembly languages, OOP, correctness, testing, and verification**.

# Week 3 — Computer Architecture and Systems

WGU’s Computer Architecture course covers **computer system design, hardware/software layouts, evolution of architecture, cross-platform computing solutions, and device integration**.

# Week 4 — Unix/Linux

WGU’s Unix/Linux course expects common Unix commands, shell navigation, file/directory management, shell configuration, and shell scripts to automate routine tasks.

# Week 5 — AI/ML Foundations + GRC

WGU’s AI/ML course includes AI/ML capabilities, applying algorithms to real problems, ethics, optimization, and data preparation. GRC covers risk frameworks, compliance, data classification, controls, audits/remediation, and risk management plans.

# Week 6 — Network Architecture, Cloud, and Distributed Systems

WGU’s Network Architecture and Cloud course covers cloud models, network systems, cloud environments, cloud-based solutions, network security, scalable distributed systems, and edge computing.

# Week 7 — Software Architecture + Mobile/Ubiquitous Design

WGU’s Software Architecture course covers architecture types, design patterns, reliable/secure systems, implementing architecture solutions, and justifying architectural choices. Mobile/Ubiquitous Design covers sensors, processors, embedded systems, communication protocols, connectivity, UX methods, privacy, scalability, and multi-device synchronization.

# Week 8 — Emerging Computing Systems + Final Integration

WGU’s Emerging Computing Systems course covers advanced topics across hardware/OS, networking/distributed systems, cloud/data centers, system design, security, performance/scalability, energy efficiency, and emerging technologies.
